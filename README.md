## 🚀 Spring Boot In-Memory CRUD API

This is a basic Spring Boot project that demonstrates how to build REST APIs without using any database.
Instead of a database, this project uses Java Map (in-memory storage) to perform CRUD operations.

## 📌 Features

Create, Read, Update, Delete (CRUD) operations<br>
RESTful API design<br>
No database dependency<br>
Simple and beginner-friendly project<br>
Exception handling included
